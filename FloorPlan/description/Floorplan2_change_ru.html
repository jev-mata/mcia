Версия 2.2

  • Добавил панель с расширенными настройками, в которых можно отключить/включить разные опции, изменить некоторые настройки проекта.

  • Добавил отключаемую опцию - Скрывать окно план, если нет открытой панорамы на плане, показывать окно плана если на нем есть текущая панорама.
  • Перенес действие создания меток компаса в файл style_Flpl.xml (открытый код) можете настроить на свое усмотрение.
  • Исправил проблему ввода цвета в строке редактора.
  • combobox.xml - полностью не раскрывался список планов(обрезАлся) - закомментировал строку 440 sub(openheight, max_overflow);
01.11.2023

Версия 2.1

  • Плагин использует для изображений плана - слои.
  • Установка в редакторе ширины и высоты, значений x y окна с планом.
  • Высота окна плана устанавливается автоматически пропорционально ширине плана(отключаемая опция)
  • Если планов несколько, то можно установить ширину окна для каждого плана индивидуально.
  • План раскрывается на установленную ширину но не больше ширины экрана
  • Можно визуально изменять масштаб радара и сохранять изменения для кажого плана.
  • Установка закрытия плана, если панорамы нет на планах, и показ плана, если текущая панорама присутствует на плане.
  • Возможность вынести выпадающий список планов отдельно и настроить его дизайнайн.
  • Возможность создать свои кнопки и назначить им открытия нужных планов.
Модуль навигации является платным дополнением для тех кто уже приобрел плагин. Входит в состав плагина для тех кто приобретает плагин:
  • Автоматическая расстановка стрелок по созданным точкам.
  • Ручной выбор стрелок переходов.
  • Настройка размера, высоты положения и наклона стрелок.
  • Возможность коррекции положения каждой стрелки.
  • Навигация по клику в сторону стрелки.
  • Возможность визуально изменять масштаб радара и сохранять изменения для кажого плана.
  • Возможность заменить картинку стрелки и настроить свой стиль для стрелок.

Версия 1.48

  • plan_settings.url_plan="%FIRSTXML%/floorplan_SM/plan/"- добавлена возможность изменять путь папки с планами.
